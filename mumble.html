Introduction

To chat in shkolopolis we are using VoIP client Mumble.

Mumble is an open source, low-latency, high quality voice chat software primarily intended for use while gaming.

How to join

  1. Download and install official client
  2. Configure sound
  3. Connect to mumble server
    • Server: shkolopolis.com
    • Port: 64738
